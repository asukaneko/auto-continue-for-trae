# 自动继续点击脚本 (Auto Clicker)

这是一个轻量级的 Python 自动化脚本，专为解决“网页/程序长时间运行后出现暂停提示”而设计。它可以自动识别屏幕上的特定提示（如“点击继续”），并在识别成功后自动点击，同时还能防止电脑自动休眠或熄屏。

## 功能特点

1.  **自动识别并点击**：使用图像识别技术（基于 OpenCV）实时扫描屏幕，发现目标图片后自动点击。
2.  **智能点击位置**：针对横幅式提示框，支持点击识别区域的右侧（例如按钮位于提示条最右侧的情况），避免误点文字。
3.  **防止系统休眠**：脚本运行时会自动调用 Windows API 保持系统唤醒，防止笔记本因长时间无操作而熄屏或睡眠。
4.  **防误触机制**：点击成功后会自动暂停 5 秒，防止重复点击。
5.  **一键启动**：提供 `run.bat` 批处理文件，双击即可运行。

## 快速开始

### 1. 环境准备

确保你的电脑已安装 Python。如果没有安装，请先下载并安装 Python。

### 2. 安装依赖

首次使用前，请运行以下命令安装所需的 Python 库：

```bash
pip install -r requirements.txt
```

或者直接双击运行目录下的 `run.bat`，它会自动尝试运行（如果环境配置正确）。

### 3. 准备截图 (重要!)

脚本依靠图像识别工作，因此你需要提供一张“目标图片”。

1.  当屏幕上出现需要点击的提示（例如“模型思考次数已达上限，请点击继续”）时，使用截图工具（如微信截图 `Alt+A` 或 Windows 自带截图 `Win+Shift+S`）。
2.  **截取包含“继续”按钮的提示框区域**。建议截取整个横条，因为脚本已配置为点击右侧。
3.  将截图保存为 **`target.png`**，并放在脚本同一目录下（覆盖原有的 `target.png`）。

### 4. 运行脚本

双击目录下的 **`run.bat`** 文件即可启动。

脚本启动后会显示：
```
系统唤醒设置成功！开始自动监控屏幕...
[14:30:00] 正在监控中...
```

### 5. 退出脚本

在脚本运行窗口中，按键盘上的 **`q`** 键即可安全退出脚本并恢复系统休眠策略。

## 高级配置

如果需要调整脚本行为，可以用记事本或代码编辑器打开 `auto_clicker.py` 修改以下参数：

*   **`CLICK_OFFSET_X`**: 点击位置的横向偏移量。
    *   `0.5`: 点击图片正中心。
    *   `0.95`: 点击图片最右侧（默认值，适用于按钮在右侧的横幅）。
*   **`CHECK_INTERVAL`**: 扫描间隔时间（默认 2 秒）。
*   **`CONFIDENCE`**: 图像匹配的相似度阈值（默认 0.8，范围 0.1-1.0）。如果识别不准，可以适当调整。

## 常见问题

*   **Q: 脚本一直提示“正在监控中...”但点击不了？**
    *   A: 请检查 `target.png` 是否清晰，或者尝试重新截图。截图范围不要太大，也不要太小。
*   **Q: 点击位置不对？**
    *   A: 修改 `auto_clicker.py` 中的 `CLICK_OFFSET_X` 参数。
*   **Q: 运行报错 `ModuleNotFoundError`？**
    *   A: 请确保已运行 `pip install -r requirements.txt` 安装依赖。
